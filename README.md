# Enhanced Fishing Bot — Downloads

Official downloads for the **Enhanced Fishing Bot**.

## Two versions
- **EnhancedFishingBot-Full.zip** — Full version (lure + consumables automation)
- **EnhancedFishingBot-Basic.zip** — Basic version (without lure automation)

Both include the BobberAssist addon. **One license key works for both.**

Grab them from the **[Releases](https://github.com/Nythrand/EnhancedFishingBot-Downloads/releases/latest)** section (right side of this page).

## How to install
1. Download the version you want from the latest release.
2. Right-click the zip → **Extract All** (don't run it from inside the zip).
3. Copy the **BobberAssist** folder into:
   `World of Warcraft\_retail_\Interface\AddOns\`
4. Run the launcher inside:
   - **Full:** `Launch Fishing Bot.bat`
   - **Basic:** `cast.bat` (with jumps) or `cast_no_jumps.bat` (no jumps)
5. Paste your license key when asked.

## Notes
- Your key locks to **one PC** at a time. Switching computers? Use the **🔄 Reset my PC** button in the Discord — instant, no waiting.
- Windows Defender may flag the download. This is a **known false positive** for game-automation tools, not a virus. See **#faq-and-install** in the Discord for the 1-minute fix.
