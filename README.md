# Enhanced Fishing Bot - Downloads

Official download for the **Enhanced Fishing Bot**.

## Download
Grab the latest version from the **[Releases](https://github.com/Nythrand/EnhancedFishingBot-Downloads/releases/latest)** section on the right.

## How to install
1. Download the zip from the latest release
2. Right-click it -> **Extract All** (don't run it from inside the zip)
3. Open the folder -> double-click **Launch Fishing Bot.bat**
4. Paste your license key when asked

## Need a license key?
You get your key from the **Get your license** button in our Discord:
**https://discord.gg/rF3H4yECc**

The bot is locked to one PC per key. Changed computers? Open a ticket in the Discord.
